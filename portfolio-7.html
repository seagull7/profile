BeerBud

Description

  • Me, Travis, Eric, Traci, Matt

  • beerbud.surge.sh

  • October, 2018

In this project my team and I were given an existing project by Timbergrove LLC and asked to finish it. The goal of the project was to make a web app that could connect to a users beer tap in order to monitor it, lock/unlock it, schedule events with it, and allow for multiple levels of users. We were given the hardware and some rough code, and were able to flush it out to a full app within a week. The system communicated to the hardware with IoT Watson, utilizes PostgreSQl on the backend and React/Redux on the front end.
